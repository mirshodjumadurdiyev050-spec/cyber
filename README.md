# Cryptographic Laboratory V2.0 🛡️

Akademik tadqiqotlar va xavfsiz ma'lumotlar almashinuvi uchun mo'ljallangan zamonaviy kriptografiya platformasi.

## ✨ Imkoniyatlar
- **Klassik Shifrlar:** Sezar, Vijiner, Vernam (XOR) vizualizatsiyasi bilan.
- **RSA Tizimi:** Professional OAEP padding va CRT tezlashtirish bilan.
- **Xesh Funksiyalar:** SHA-256 Avalanche (ko'chki) effekti tahlili.
- **Akademik PDF:** Har bir bo'lim uchun professional qo'llanmalarni yuklab olish.
- **Modern UI:** Light mode, glassmorphism va interaktiv animatsiyalar.

## 🚀 Ishga tushirish
1. Kutubxonalarni o'rnating: `pip install -r requirements.txt`
2. Dasturni ishga tushiring: `streamlit run main.py`

## 🛠️ Texnologiyalar
- Python 3.x
- Streamlit
- FPDF2
- Pandas

## 🎓 Muallif
Ushbu loyiha kiberxavfsizlik asoslarini o'rganuvchilar uchun yaratilgan.
